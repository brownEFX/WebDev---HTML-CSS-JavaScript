CSS Web Safe Fonts

	

		**Web safe fonts are fonts that are universally installed across all browsers and devices.
	
	
	

Fallback Fonts

	

	**Use Fallback Fonts in case a font is not found.
	**Add a list of similar "backup fonts" in the font-family property. 
	**If the first font does not work, the browser will try the next one, and the next one, and so on. 
	**Always end the list with a generic font family name.
	**Syntax:
	
	p {
	/*font-family: FontOption1, FontOption2, Generic Font family;*/
	font-family: Tahoma, Verdana, sans-serif;
	}
