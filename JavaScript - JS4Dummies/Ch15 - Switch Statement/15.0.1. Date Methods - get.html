JavaScript Date Methods

JavaScript Date Methods
Method What is does
Date - get Methods:
getDate() Gets the day of the week as a number (0-6).
getDay() Concatenates two or more strings.
getFullYear() Gets the year (yyyy).
getHours() Gets the hour (0-23).
getMilliseconds() Gets the fraction of a second (0-999).
getMonth() Gets the month (0-11).
getSeconds() Gets the seconds (0-59).
getTime() Gets the time, in milliseconds (Unix).
