JavaScript Array Methods

arrayMethods
Method Description
concat() A new array made up of the current array, joined with other array(s) and/or value(s).
join() Joins all the elements of an array into a string using default or specified separator.
indexOf() Returns the first occurrence of the specified value or –1 if the value is not found.
lastIndexOf() Returns the last occurrence of the specified value or –1 if the value is not found.
push() Adds new items to the end of an array.
pop() Removes the last element in an array.
unshift() Adds new items to the end of an array. Returns new array && array.length.
shift() Removes the first element from an array and returns that element, resulting in a change in length of an array.
reverse() Reverses the order of elements in an array.
sort() Sorts array alphabetically, in ascending order. Returns an array in new sort order
slice() Selects a portion of an array and returns it as a new array.
splice() Returns a new array comprised of elements that were added to or removed from a given array.
toString() Converts an array to a string.
forEach() Calls a function for each element in an array.
