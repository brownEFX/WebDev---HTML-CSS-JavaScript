Table Padding

	

	**To control the space between the border and the content in a table, use the padding property on /td> and /th> elements:
	
	Example 1 - left, right, or center alignment
			
		/style>
			table, td, th {
			  border: 1px solid black;
			}
			table {
			  border-collapse: collapse;
			  width: 100%; /*To create a table that should only span half the page, use width: 50%*/
			}
			th {
			  height: 70px;
			}
			td {
				text-align: left;
				text-align: center;
				text-align: right;
			}
			th, td  {
				padding: 15px; /*This property adds space between the border and the content in a table.*/
				
				text-align: left;
				text-align: center;
				text-align: right;
			}
		/style>

Striped Tables

	

	**For zebra-striped tables, use the nth-child() selector and add a background-color to all even (or odd) table rows:
	
	Example 1 - left, right, or center alignment
			
		/style>
			table, td, th {
			  border: 1px solid black;
			}
			table {
			  border-collapse: collapse;
			  width: 100%; /*To create a table that should only span half the page, use width: 50%*/
			}
			tr:hover {
				background-color: coral; /*Move the mouse over the table rows to see the effect.*/
				}
			tr:nth-child(even) {
				background-color: #f2f2f2;
				}
			th {
			  height: 70px;
			}
			td {
				text-align: left;
				text-align: center;
				text-align: right;
			}
			th, td  {
				border-bottom: 1px solid #ddd; /*Add the border-bottom property to th and td for horizontal dividers*/
				 
				padding: 15px; /*This property adds space between the border and the content in a table.*/
				
				text-align: left;
				text-align: center;
				text-align: right;
			}
		/style>
