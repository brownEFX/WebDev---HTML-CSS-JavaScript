JavaScript Built-in Functions

jsFunctions
Function Description
String Functions:
charAt(index) Returns the character at the specified index.
concat(str1, str2, ...) Concatenates two or more strings.
toUpperCase() Converts a string to uppercase.
toLowerCase() Converts a string to lowercase.
Array Functions:
join() Joins all the elements of an array into a string using default or specified separator.
push() Adds new items to the end of an array.
pop() Removes the last element in an array.
unshift() Adds new items to the end of an array. Returns new array && array.length.
shift() Removes the first element from an array and returns that element, resulting in a change in length of an array.
Math Functions:
Math.random() Returns a random number between 0 and 1.
Math.round() Round a number to the nearest integer, round down, or round up.
Math.floor() Round a number down to the nearest integer.
Math.ceil() Round a number up to the nearest integer.
Date Functions:
new Date() Creates a new Date object representing the current date and time.
getMonth() Retrieve the month from a Date object.
getDate() Retrieve the month from a Date object.
getFullYear() Retrieve the year from a Date object.
DOM Manipulation Functions:
document.getElementById() Gets an HTML element by its ID.
document.getElementByClassName() Gets an HTML element by their class name.
document.getElementByTagName() Gets an HTML element by their tag name.
Even Handling Functions:
addEventListener(event, function) Attaches an event handler to an element.
removeEventListener(event, function) Removes an event handler to an element.
JSON Functions:
JSON.parse Parses a JSON string and returns a JavaScript object.
JSON.stringify Converts a JavaScript object to a JSON string.
