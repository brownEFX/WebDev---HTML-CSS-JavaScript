p Element

The HTML p element defines a paragraph. A paragraph always starts on a new line and has margin before and after.

em Element

em elements Adds emphasis to a word or words. Typically displayed as italics in web browsers.

strong Element

The strong tag is used to define text with strong importance. The content inside is typically displayed in bold.
