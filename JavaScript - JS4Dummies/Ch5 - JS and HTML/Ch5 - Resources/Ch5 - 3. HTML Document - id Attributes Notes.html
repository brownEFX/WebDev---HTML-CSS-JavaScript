getElementById

getElementById locates/select an individual element in a document so that you can change it, delete it, or add something to it.

    getElementById - Steps
  1. Ensure that the element you want to select has an id attribute.
  2. Locate that element: document.getElementById("id‐value")
Example

To select an element with an id value of myName: document.getElementById("myName")

Getting what’s inside an element with innerHTML

innerHTML is a property of every element. A property describes an aspect of an object - something that an object has. innerHTML tells you what’s between the starting and ending tags of the element. innerHTML lets you set the contents of the element. Example
