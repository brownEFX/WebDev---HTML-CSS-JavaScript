HTML Elements
Element Description Usage
HTML element: Root of an HTML document Container for all other elements
Head element: Container for metadata Title, Style, Base, Link, Meta, Script, NoScript
Body element: Defines the document's body. H1-h6, paragraphs, images, hyperlinks, tables, lists
H1-h6 Element Defines HTML headings. h1 = most important, h6 = least important.
ol | ul Element ol = ordered list of numbered items. ul = unordered list of unnumbered items.
li Element li = list item used inside ol | ul Element
em Element Emphasis Adds emphasis to a word as italics.
strong Element Strong Strong importance to a word as bold.
a Element Anchor A Link.
img Element Image An image.
div Element Division Separates a document into different parts.
