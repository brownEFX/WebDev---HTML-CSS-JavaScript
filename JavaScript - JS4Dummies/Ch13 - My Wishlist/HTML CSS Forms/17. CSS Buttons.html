Button Sizes


			Use the font-size property to change the font size of a button:

Disabled Buttons


			Use the opacity property to add transparency to a button (creates a "disabled" look).
			Tip: You can also add the cursor property with a value of "not-allowed", which will display a "no parking sign" when you mouse over the button:
Example - Disabled Buttons /style> body { color: black; font-family: Helvetica, Verdana, sans-serif; font-size: 16px; background-image: url("paper.gif"); style:text-align:center; } .button { background-color: #4CAF50; /* Green */ border: none; border-radius: 4px; width: 120px; color: white; padding: 15px 32px; text-align: center; text-decoration: none; display: inline-block; margin: 4px 2px; cursor: pointer; border: 2px solid #e7e7e7; transition-duration: 0.4s; } .disabled { opacity: 0.6; cursor: not-allowed; /style>
